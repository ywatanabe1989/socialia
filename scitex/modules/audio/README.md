# scitex.audio - Social Content

## Overview
Text-to-speech MCP server with queue management.

## Key Features
- Multiple TTS backends (pyttsx3, gtts, elevenlabs)
- Automatic fallback
- Speech queue for sequential playback
- WSL audio support

## Target Audience
- Primary: AI/LLM developers
- Secondary: Accessibility developers

## Differentiators
- MCP-native speech for AI agents
- Handles WSL audio quirks
- Queue prevents audio overlap
