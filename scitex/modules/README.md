# SciTeX Modules - Social Outreach

Organize social content by SciTeX module for targeted promotion.

## Module Structure

```
modules/
├── ai/          # AI integration, LLM tools
├── audio/       # Audio processing, MCP server
├── browser/     # Browser automation
├── capture/     # Screenshot capture MCP
├── db/          # Database utilities
├── fig/         # Plotting, visualization
├── io/          # File I/O utilities
├── scholar/     # Academic research tools
├── stats/       # Statistical analysis
├── tex/         # LaTeX integration
├── torch/       # PyTorch utilities
└── writer/      # Document generation
```

## Per-Module Content

Each module directory can contain:
- `README.md` - Module description, key features, target audience
- `drafts/` - Draft posts for various platforms
- `posted/` - Published content with metrics
- `assets/` - Screenshots, GIFs, diagrams

## Priority Modules for Promotion

| Module  | Audience              | Key Platforms       |
|---------|-----------------------|---------------------|
| fig     | Data scientists       | Reddit, Twitter     |
| capture | Developers            | Twitter, GitHub     |
| audio   | Developers            | Twitter, GitHub     |
| scholar | Researchers           | LinkedIn, Medium    |
| stats   | Researchers, analysts | LinkedIn, Medium    |
| ai      | ML engineers          | Twitter, Reddit     |

## Cross-Module Campaigns

Some posts can highlight module integration:
- `fig` + `stats` → Statistical visualization
- `scholar` + `tex` → Academic writing workflow
- `capture` + `audio` → MCP ecosystem
