# Twitter/X 投稿フォーマット (日本語)

## 構造 (~280文字)

```
[WHY]   1行 - 普遍的な気づき、パンチのあるフック
        ↓
[PAIN]  暗示または1行
        ↓
[CODE]  最大4-6行
        ↓
[HOW]   1行まとめ
        ↓
[CTA]   pip install + ハッシュタグ
```

## テンプレート

```
[物事がこうあるべきという普遍的な気づき]

[なぜXではそうなっていないのか？]

\`\`\`python
# 最小限のデモ
\`\`\`

[一行で説明]

pip install scitex[module]

#Python #Research #OpenSource
```

## 制約

- コードブロック除き280文字以内
- コード: 最大4-6行
- 説明不要 - コードが語る
- 疑問形で注目を引く
