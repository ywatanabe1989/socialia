AI agents can now create publication-ready scientific figures.

FigRecipe v0.14.0 introduces an MCP server that lets AI assistants plot, compose, and reproduce figures autonomously.

What this means in practice:
- Describe what you want in natural language
- Agent creates the figure with proper styling
- Everything is reproducible via YAML recipes
- Full data extraction and validation built-in

Installation:
```

```
