# SciTeX Social

Outreach management for [SciTeX](https://scitex.ai) - an open-source ecosystem for scientific research workflow.

## Structure

```
scitex/
├── README.md
├── assets/               # Shared images, GIFs, demos
├── drafts/               # Work in progress
│   └── YYYY-MMDD-platform.md
└── posted/               # Published posts
    └── YYYY-MMDD-platform/
        ├── YYYY-MMDD-platform.md
        ├── assessment.md
        └── insights-*.png
```

## Key Messages

- **Tagline**: Science × LaTeX - infrastructure for reproducible research
- **Components**: Scholar (literature), Vis (figures), Code (analysis), Writer (manuscripts)
- **Value prop**: Efficiency, reproducibility, verifiability in scientific research
