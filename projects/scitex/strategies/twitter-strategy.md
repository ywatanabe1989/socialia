# SciTeX Twitter Strategy (@SciTeX_AI)

## Daily Format: "Today's Progress"

Simple, authentic, sustainable.

```
Today's Progress on SciTeX:

- [What you did today]
- [Another thing]

[Optional one-liner reflection]

#BuildInPublic #OpenScience
```

## Guidelines
- Post what you actually did
- Keep it real (failures count too)
- 2-4 bullet points max
- Add image/screenshot when relevant

## Hashtags
`#BuildInPublic #OpenScience`

Optional: `#ResearchTools #LaTeX #DataViz #PhD`

## Tracking
Log each post in `posted/YYYY-MMDD-twitter/`
