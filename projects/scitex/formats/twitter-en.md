# Twitter/X Post Format (English)

## Structure (~280 chars)

```
[WHY]   1 line - Universal insight, punchy hook
        ↓
[PAIN]  Implied or 1 line
        ↓
[CODE]  4-6 lines max
        ↓
[HOW]   1 line summary
        ↓
[CTA]   pip install + hashtags
```

## Template

```
[Universal observation about how things should work]

[Why doesn't X work this way?]

\`\`\`python
# minimal demo
\`\`\`

[What it does in one line]

pip install scitex[module]

#Python #Research #OpenSource
```

## Constraints

- Under 280 characters (excluding code block)
- Code: 4-6 lines max
- No explanations - code is the message
- Question format grabs attention
