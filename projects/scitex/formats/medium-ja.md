# Medium 投稿フォーマット (日本語)

## 構造

```
タイトル: [キャッチーだが誠実 - 釣りタイトル禁止]
サブタイトル: [一行の価値提案]

[WHY]   冒頭のフック - 普遍的な気づき
        ↓
[PAIN]  問題 - 共感できる不満
        ↓
[HOW]   解決策 - 「だから作った」
        ↓
[CODE]  複数の例、コメント付き
        ↓
[FEATURES]  詳細な説明
        ↓
[CLOSING]  個人的な振り返り + CTA
```

## テンプレート

```
# [問題についてのキャッチーなタイトル]
## [解決策の一行説明]

[普遍的な気づき - 読者がうなずく1-2文]

[不便さ - なぜこれが面倒なのかを展開。個人的な視点がMediumでは効果的]

[解決への移行 - 「だから〜することにした」「N回目に同じコードを書いた後...」]

## 解決策
```

## 制約

- Medium読者は洗練された文章とストーリーを期待
- コード例は包括的だが圧倒的にならないように
- 個人的なストーリーが効果的
- 「10倍」「革新的」などの言葉を避ける
- 前向きな締めくくりで終わる（SciTeXへの布石）
- タグ: Python, Programming, Data Science, Research, Open Source
