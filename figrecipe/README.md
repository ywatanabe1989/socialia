# scitex-social (Private)

Personal repository for managing SNS outreach for SciTeX/FigRecipe projects.

## Structure

```
scitex-social/
├── drafts/          # Post drafts (1 post = 1 file)
├── assets/          # Images, GIFs for posts
├── logs/            # Reaction tracking
└── README.md
```

## Workflow

1. Write draft in `drafts/YYYY-MM-DD-platform-topic.md`
2. Use AI to adapt for different platforms
3. Post and log in `logs/`

## Targets

| Platform          | Style                    | Notes                     |
|-------------------|--------------------------|---------------------------|
| r/dataisbeautiful | Visual-heavy, short text | Good for polished figures |
| r/visualization   | Mixed technical/visual   | Current home              |
| r/dataengineering | Reproducibility focus    | Emphasize workflow        |
| Hacker News       | Technical, clear README  | Wait until polished       |
| Twitter/X         | 1 image + 1 line         | #DataViz #OpenScience     |

## Rules

- Stagger posts by hours/days
- Change 30-50% of text per platform
- Never copy-paste identical content
